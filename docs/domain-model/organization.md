# Organization

An organization represents an entity that creates and manages schedules.

Examples include:

* a university;
* a driving school;
* a sports club;
* an association;
* a company;
* a training center.

An organization can contain members, schedules, and organization-level settings.

Each organization has an authoritative timezone.

## Business Rules

* Every organization must define a timezone.
* The organization timezone is used as the default timezone for schedules created within the organization.
* The organization acts as the main boundary for membership, access control, and schedule ownership.
